System Architecture
01 — EXTRACT
Canvas API samples colors, brightness, saturation, edge density and warmth from the uploaded image pixel data.
02 — HASH
SHA-256 cryptographic fingerprint generated from image data. Unique to every pixel arrangement. Unforgeable.
03 — COMPARE
Cosine similarity on color vectors + shape/composition matching — same principle as ORB feature matching in OpenCV.
04 — AI JUDGE
Claude AI reads the extracted features and writes a unique, human-readable originality report streamed live.
05 — SCORE
Weighted formula: 70% color similarity + 15% shape match + 15% composition. Inverted to get originality %.
06 — SEAL
Proof-of-Originality certificate issued on simulated blockchain with block number, TX hash and timestamp.
