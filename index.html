System Architecture
01 — EXTRACT
Canvas API samples colors, brightness, saturation, edge density, and warmth from the uploaded image.
02 — HASH
SHA-256 fingerprint generated from image data — unique to each pixel arrangement, unforgeable.
03 — COMPARE
Cosine similarity on color vectors + shape/comp matching, similar to ORB feature matching in OpenCV.
04 — AI JUDGE
Claude AI reads extracted features and writes a human-readable originality report with recommendations.
05 — SCORE
Weighted formula: 70% color similarity + 15% shape match + 15% composition. Inverted to originality %.
06 — SEAL
Certificate issued on simulated blockchain with block number, timestamp, score, and SHA-256 hash.
